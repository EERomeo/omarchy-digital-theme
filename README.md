# Installation
You can install this theme by running in the terminal:

```omarchy-theme-install https://github.com/EERomeo/omarchy-digital-theme```

or select Install > Style > Theme via the Omarchy Menu and provide the link ```https://github.com/EERomeo/omarchy-digital-theme```
